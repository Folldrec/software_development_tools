Основний функціонал:

Розріджені структури даних

SparseList - розріджений список, що зберігає лише ненульові елементи
SparseMatrix - розріджені матриці в форматах COO (Map-based) та CSR
Операції: додавання, множення, транспонування матриць, множення матриці на вектор
Пошук елементів за значенням та за заданою умовою
Доступ за індексом, підрахунок ненульових елементів
Збереження та завантаження з файлів

Математичний аналіз

Рівень A (базовий):

Послідовності: арифметичні, геометричні, рекурсивні (Фібоначчі), функціональні
Функції однієї змінної: поліноми, тригонометричні, експоненційні, логарифмічні
Обчислення значень функцій у точках

Рівень B (алгоритми):

Символьне обчислення похідних довільного порядку
Чисельне інтегрування методом трапецій
Розклад функцій в ряди Тейлора
Обчислення границь послідовностей
Обчислення сум рядів та часткових сум
Пошук коренів функцій методом Ньютона

Рівень C (інтеграція):

Експорт функцій у формат Wolfram Mathematica
Експорт у формат Python (SymPy)
Експорт у формат LaTeX для наукових публікацій
Табулювання функцій з експортом у файли
